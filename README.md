# Proxima

**Proxima** - мощный и точный инструмент для математической аппроксимации функций, построенный на самом современном стеке технологий Android с поддержкой IOS.

## Возможности
*   **Точные вычисления:** Использование библиотеки `BigDecimal` для минимизации погрешностей при расчетах.
*   **6 моделей аппроксимации:** Линейная, квадратичная, кубическая, экспоненциальная, логарифмическая и показательная функции.
*   **Умный выбор:** Автоматическое определение наилучшей модели на основе коэффициента детерминации ($R^2$).
*   **Интерактивные графики:** Визуализация данных и полученных кривых в реальном времени.
*   **LaTeX рендеринг:** Отображение итоговых формул в классическом математическом виде.
*   **Экспорт:** Возможность копирования полученных функций в буфер обмена.

## Технологический стек
*   **UI:** `Jetpack Compose` - современный декларативный интерфейс.
*   **Навигация:** `Voyager` - многоплатформенная библиотека навигации.
*   **Графики:** `Canvas` - стандартная библиотека для работы с графика.
*   **Архитектура:** `MVVM` + `UDF` - для реактивной отзывчивости приложения.
*   **Математика:** `Kotlin BigNum` - библиотека для высокоточных вычислений.
*   **Рендеринг:** `LatexRenderer` - для отображения формул.

## Пример данных
Для проверки работы введите следующие точки:
`(1, 2), (2, 3), (3, 4)`

**Результат:** Линейная аппроксимация $f(x) = 1x + 1$ с $R^2 = 1.0$.
